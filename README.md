## 🔍 Что сделано в проекте

### 1. Предобработка и очистка данных
Исходная выгрузка содержала типичные проблемы реальных данных:
- Удалены знаки валют (`$`) из числовых полей `cost` и `sale_amount`, выполнена типизация
- Исправлены опечатки в названиях кампаний, городах, устройствах и ключевых словах через словари замен
- Разобраны смешанные форматы дат (`YYYY-MM-DD` и `DD-MM-YY`)
- Восстановлены пропущенные значения расчётных метрик — **100% строк сохранены**

### 2. Исследовательский анализ (EDA)
- Распределения ключевых метрик: показы, клики, затраты, выручка, конверсия
- Корреляционная тепловая карта по всем числовым признакам
- Временные ряды: динамика показов и кликов за ноябрь 2024
- Сравнение конверсии в разрезе устройств (boxplot)

### 3. Статистическое тестирование гипотез

| Гипотеза | Тест | Результат |
|---|---|---|
| Conversion Rate различается по устройствам | Тест Левена + Краскел–Уоллис | Различий нет (p > 0.05) |
| CTR различается по ключевым словам | Тест Левена + Краскел–Уоллис | Различий нет (p > 0.05) |
| ROAS различается по ключевым словам | Тест Левена + однофакторный ANOVA | Различий нет (p > 0.05) |
| Метрики различаются по дням недели | Тест Левена + Краскел–Уоллис | Различий нет (p > 0.05) |

### 4. Анализ воронки продаж
Построена воронка Impressions → Clicks → Leads → Conversions с расчётом конверсии на каждом переходе.

### 5. Анализ недельной цикличности
Рассчитаны средние значения кликов, затрат, конверсии, выручки и ROAS по дням недели с визуальной разметкой лидеров.

## 💡 Ключевые находки

**1. Главное узкое место воронки — переход «Клики → Лиды» (конверсия 14.4%)**  
При этом отдел продаж работает эффективно: конверсия из лида в покупку — 32.7%. Сайт теряет 85% заинтересованного трафика ещё на этапе первого контакта.

**2. Устройства и ключевые слова не влияют на эффективность**  
Conversion Rate, CTR и ROAS статистически одинаковы во всех сегментах. Перераспределение бюджета между устройствами или ключевыми словами без дополнительных тестов не обосновано.

**3. Суббота — наиболее выгодный день**  
При минимальных затратах достигается максимальный ROAS (7.18). Понедельник даёт пик кликов (в среднем 142). Несмотря на это, статистически кампания работает стабильно вне зависимости от дня недели.

**4. Волатильность во второй половине ноября**  
После 15 ноября показатели стали выше, но менее предсказуемыми — стоит отслеживать в следующих периодах.

## 🧪 Предложенные A/B-тесты

**Тест 1. Упрощение лендинга**  
Гипотеза: форма с одним экраном и двумя полями (имя + почта) увеличит конверсию Клики → Лиды.  
Метрика: `Leads / Clicks`

**Тест 2. Заголовки с глаголом-намерением**  
Гипотеза: заголовки «Стань аналитиком данных» / «Освой профессию с нуля» дадут более высокий CTR, чем текущие информационные формулировки.  
Метрика: `CTR`
